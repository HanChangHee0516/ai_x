1. 자료형
-. 문자열(string), 숫자(number), 불리언(boolean)
-. 객체(object: 배열 포함), 함수(function), undefined

2. 변수 선언 키워드
-. var: 재선언 가능, 함수 범위
-. let: 재선언 불가, 블록 범위
-. const: 상수, 재할당 불가 (객체 내부 속성은 변경 가능)

3. 표현식 vs 문장
-. 표현식: 값으로 평가될 수 있는 식
-. 문장: 실행 가능한 코드 단위, 끝에 ; 사용

4. 식별자 규칙
-. 키워드 사용 금지, 숫자로 시작 불가
-. 특수문자 _, $만 허용, 공백 금지
-. 관례: 생성자 함수는 대문자 시작, 변수/함수는 소문자 시작

5. 주석
-. 한 줄: //
-. 여러 줄: /* ... */

6. 출력/대화상자 함수
-. alert() : 알림 창
-. confirm() : 확인/취소 반환
-. prompt() : 사용자 입력 받기
-. document.write() : HTML 출력
-. console.log() : 개발자 콘솔 출력

7. 자료형 변환
-. Number(), parseInt(), parseFloat(): 숫자로
-. String(): 문자열로
-. Boolean(): true/false
-. isNaN(): 숫자인지 판별

false로 간주되는 값: 0, '', NaN, null, undefined, false

8. 연산자
-. 산술: +, -, *, /, %
-. 비교: ==, ===, !=, !==, >, <, >=, <=
-. 논리: &&, ||, !
-. 복합 대입: +=, -=, *=, /= 등
-. 증감: ++, --

9. 문자열 관련
-. ', " 모두 사용 가능 (혼용 주의)
-. 이스케이프 문자: \\, \", \', \n 등

10. 일치 연산자
-. ==: 값만 비교 (형 변환 포함)
-. ===: 값 + 자료형 모두 비교
