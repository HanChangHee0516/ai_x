
JavaScript 개요 요약 정리본

1. 추천 튜토리얼/학습 사이트
- https://javascript.info (모던 JS 튜토리얼)
- https://developer.mozilla.org/ko (MDN 문서)
- https://www.w3schools.com (기초 튜토리얼)

2. 자바스크립트의 역사
- 넷스케이프사에서 "모카(Mocha)"라는 이름으로 시작
- Java와 이름만 유사할 뿐, 완전히 다른 언어
- 웹 페이지를 정적으로 보여주던 초기 시대에서 동적 기능을 구현하는 핵심 도구로 발전

3. 자바스크립트의 역할
- HTML 안에 포함되어 동적인 웹 페이지 구현
- 사용자 인터랙션 (클릭, 입력 등)에 반응
- 웹 애플리케이션으로 진화한 웹 문서의 중심 기술

4. script 태그 위치에 따른 실행 시점
- <head>에 넣으면 본문(body)보다 먼저 실행됨
- <body>에 넣으면 본문이 먼저 렌더링된 후 실행됨
- 대부분의 경우, script는 head에 넣는 것이 가독성과 관리에 유리

5. 예제 요약: document.write()
- document.write("<h2>메세지</h2>") → HTML에 텍스트 직접 삽입
- 예시 코드에서 여러 줄의 메시지를 HTML에 출력하는 방식 연습

6. 예제 요약: innerHTML, alert
- document.body.innerHTML += ... : 기존 body에 새로운 내용 추가
- alert('메시지') : 알림창 출력
- 잘못된 변수 연산 예시 (문자/숫자 혼용) → 오류 유발

7. 오류 확인 방법
- 크롬 개발자 도구(F12)를 이용해 콘솔 오류 확인
- 코드 위치, 줄번호를 확인하고 디버깅 가능
- IE에서는 "스크립트 디버깅 사용 안 함" 체크 해제 필요
