
CSS 선택자 핵심 요약 정리본

1. 선택자란?
HTML 문서 내 특정 요소를 선택해 CSS 스타일을 적용하는 방식.

2. 주요 선택자 정리

- 태그 선택자(tag): 특정 태그 전체 선택 (ex: div, p 등)
- 전체 선택자(*): 문서 전체 요소 선택 (초기화 등에 사용)
- ID 선택자(#id): 특정 id 속성값을 가진 요소 선택
- 클래스 선택자(.class): 특정 class 속성값을 가진 요소 선택
- 혼합 선택자: 태그 + ID/class 조합으로 선택

3. 속성 선택자

- [속성]: 해당 속성을 가진 요소
- [속성=값]: 해당 속성과 값을 정확히 가진 요소
- [속성^=값]: 값으로 시작하는 속성을 가진 요소
- [속성$=값]: 값으로 끝나는 속성을 가진 요소

4. 관계 선택자

- 후손 선택자 (A B): A의 모든 하위 요소 B 선택
- 자손 선택자 (A > B): A 바로 아래의 자식 B만 선택
- 동위 선택자 A + B: A 바로 다음 B 하나 선택
- 동위 선택자 A ~ B: A 이후의 모든 형제 B 선택

5. 반응 선택자 (동적 상태)

- :hover: 마우스 올렸을 때
- :focus: 포커스 되었을 때 (ex: input 클릭 시)
- :enabled / :disabled: 활성화/비활성화 input 구분

6. 구조 선택자

- :first-child / :last-child: 첫 번째/마지막 자식
- :nth-child(n): n번째 자식
- :nth-child(2n+1): 홀수 번째 자식
- :not(선택자): 특정 선택자를 제외한 요소

7. 링크 선택자

- a[href]: 링크 태그 선택
- a::after { content: attr(href); } : 링크 주소를 텍스트로 출력

8. 부정 선택자

- :not(.class): 해당 class를 제외한 요소 선택

9. 예제 요약

- ul li:hover → li에 마우스를 올렸을 때 스타일 적용
- input[type='text'] → 타입이 text인 input
- div > p → div의 바로 아래 p
- h3 + div → h3 바로 다음 div
- h3 ~ div → h3 이후 모든 형제 div

10. 실전에서의 팁

- 초기화용 전체 선택자(*)는 가장 위에 작성
- class는 반복되는 요소, id는 유일한 요소에 사용
- 속성 선택자와 구조 선택자는 동적 UI에 매우 유용
